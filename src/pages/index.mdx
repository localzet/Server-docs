# Что такое Localzet Server?

Localzet Server - это высокопроизводительный сервер PHP-приложений с открытым исходным кодом. Это не MVC фреймворк, а более низкоуровневое и универсальное сервисное ПО. Его можно назвать инновацией в PHP, которая позволяет разработчикам освободиться от ограничений, связанных с возможностями PHP, ориентированными только на веб-разработку.  {{ className: 'lead' }}

## Варианты использования {{ anchor: false }}

Localzet Server отличается от традиционных фреймворков тем, что его можно использовать не только для веб-разработки, но и в более широком спектре областей, таких как сообщения в реальном времени, интернет вещей, игры, управление сервисами и другими серверами или промежуточным ПО, что значительно расширяет горизонты для PHP-разработчиков. Мы рекомендуем разработчикам не только уметь его использовать, но и способствовать развитию своих собственных open-source проектов на основе Localzet Server для улучшения своих навыков и производительности проектов.  {{ className: 'lead' }}
